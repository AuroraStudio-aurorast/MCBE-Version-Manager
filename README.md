# Minecraft 基岩版版本管理器

源项目：https://github.com/MCMrARM/mc-w10-version-launcher

## 要求

1. Windows 10及更高版本
2. 您需要先获取 Minecraft 试玩版。
3. .NET Framework 4.8

## 亮点

1. 完全汉化
2. 使用了第三方主题替代原版 WPF 主题
3. 使用 FastGit 替换了原来的 Github Raw（国内可以正常使用）
4. 集成了解除使用限制工具

## 编译

请使用 Visual Studio 2022，并包含最新的 Windows SDK！
